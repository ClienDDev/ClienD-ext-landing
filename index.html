Выставление оценок с помощью клавиатуры

После установки расширения появляется возможность ставить оценки с помощью клавиатуры.

Для начала работы нужно поставить курсор на одну из клеточек журнала. Затем нажмите на клавиатуре цифру, соответствующую оценке. Например, кнопка 5 поставит оценку "5".

После того, как оценка поставится, нажмите на ↓, чтобы спуститься вниз.

Поставим оценку 4 с помощью кнопки 4.

После завершения операции страница будет выглядеть так:

Полный список клавиш

  • 1 - поставить оценку "1"
  • 2 - поставить оценку "2"
  • 3 - поставить оценку "3"
  • 4 - поставить оценку "4"
  • 5 - поставить оценку "5"
  • 0 - убрать оценку
  • Н - поставить отметку "не был"
  • Б - поставить отметку "болел"
  • Х - убрать отметку об отсутствии

Загрузка новостей из ВКонтакте

  1. Откройте стену человека или группы
  2. Нажмите на кнопку "Поделиться" ()
  3. Нажмите на кнопку "В новости edu.tatar.ru". Откроется список новостных блоков
  4. Выберите нужный новостной блок
  5. Нажмите на кнопку "Добавить новость"
  6. Название и текст новости добавятся автоматически. Фотографию придется добавить в полуручном режиме.
  7. Добавление фотографии
    • Скопировать ссылку из получившегося окна (CTRL+C)
    • Откроется окно добавления фотографии к новости
    • Нажать на кнопку "Выбрать файл" и вставить ссылку, которую мы скопировали до этого (CTRL+V)
    • Выделить размер миниатюры
    • Сохранить
  8. Завершить добавление новости

Также на странице добавления есть кнопка "Добавить пост из ВКонтакте", которая помогает загрузить запись из ВКонтакте по ссылке.

Функционал не будет работать в школьной сети Интернет из-за того, что социальная сеть ВКонтакте там заблокирована.


Передает ли расширение мои данные третьим лицам?
Нет. Расширение не использует сторонние сервера и сервисы для работы. Все данные обрабатываются в браузере и никуда не отправляются. Вы можете убедиться в этом сами - исходный код открыт для всех.

Расширение мешает в работе. Как его удалить?

  • Зайдите в настройки Chrome (Меню/Настройки)
  • Перейдите на вкладку "Расширения"
  • Найдите расширение ClienD ext
  • Справа нажмите на кнопку "Удалить" ()

После этого расширения удаляется полностью.
